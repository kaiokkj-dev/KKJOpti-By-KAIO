🚀 KKJOpti by Kaio

Um aplicativo desktop focado em otimização de desempenho do Windows, com melhorias rápidas em FPS, sistema, internet e inicialização — tudo com poucos cliques.

🧠 Sobre o projeto

O KKJOpti foi desenvolvido para facilitar a vida de usuários que querem extrair o máximo desempenho do PC sem precisar mexer manualmente em configurações avançadas.

Com uma interface moderna e intuitiva, o app permite aplicar otimizações reais usando scripts .bat e .reg diretamente no sistema.

⚙️ Funcionalidades
🟢 Otimização geral
Limpeza de processos desnecessários
Ajustes automáticos no sistema
Melhor uso de CPU e RAM
🎮 FPS Boost
Tweaks no Windows para melhorar desempenho em jogos
Desativação de serviços que consomem recursos
Ajustes no registro para reduzir input lag
🖥️ Windows
Otimizações visuais (remoção de efeitos)
Melhor resposta do sistema
Configurações ocultas ativadas
🌐 Internet
Redução de latência
Ajustes de rede via registro
Melhor estabilidade de conexão
⚡ Inicialização
Controle de programas que iniciam com o sistema
Redução do tempo de boot
🧱 Tecnologias utilizadas
Electron.js → criação do app desktop
HTML5 + CSS3 → interface
JavaScript (Node.js) → lógica e execução
Scripts .bat / .reg → otimizações reais no sistema

▶️ Como usar
Instale as dependências:
npm install
Execute o app:
npm start
Clique nas opções desejadas e aplique as otimizações.
